Funções e Funções Afim

Noção Básica das Funções

A função é um processo que você pode fazer para transformar um número para outro valor através de alguma operação numérica para certa funcionalidade. Um exemplo de uso de uma função é para calcular o valor de um salário somente possuindo o valor das vendas mensais.

Ex: f(x) = 2400 (Valor fixo) + x (Vendas) * 0,1 (Ganho das vendas)

Numa função, será terá valores de entrada e valores de saída. Isto é um conceito semelhante que ocorre no que chamamos de métodos ou funções dentro da programação, que seria um trecho de código com geralmente parâmetros (valores de entrada) que fazem certa operação e retornam um valor específico (valores de saída). Inclusive é muito usual utilizar como prática de algoritmos, desenvolver um método na programação que faça uma máquina de funções afim. Logo veremos a definição de função afim.

Notação das Funções

X → Variável Independente
Y ou f(x) → Variável dependente
Ex: f(x) = y = 10 + 4x

Lei da Formação

Isso é como o valor será transformado, sendo f: R → R, utilizando alguma operação que transforme o valor de entrada para o valor de saída. Em outras palavras, a relação entre os termos x e y através de uma equação matemática.

Função Afim

f(x) = ax + b

Função Afim é uma das funções mais básicas da matemática. É uma função afim, toda a função que possui dois números reais a e b (são coeficientes) tal que f(x) = ax + b, para todo x ∈ ℝ.

Se não foi possível ver as equações abaixo, provavelmente seu navegador não suporta a linguagem de marcação MathML. Para solucionar, busque acessar o website em outro navegador.
Coeficiente Linear: b (valor Inicial)
b = f ( 0 )

Coeficiente Angular: a (taxa de variação)
a = f ( x 2 ) - f ( x 1 ) x 2 - x 1

Lei da Formação da Função Afim

Os dois métodos que foram abordados em aula foram o de construir uma tabela com os possíveis valores de x, e outro sendo encontrar os dois valores (x e y) como pontos de corte dos eixos (x quando y = 0 e y quando x =0).

Classificação das Funções Afim

Existem 3 classificações de uma função afim em relação a sua progressão e sua representação gráfica:

  • - a > 0 → Função Crescente
  • - a < 0 → Função Decrescente
  • - a = 0 → Função Constante
