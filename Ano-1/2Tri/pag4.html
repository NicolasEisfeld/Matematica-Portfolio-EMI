Função Logarítmica

Definição de um Logaritmo

ac = b → logab = c


Muito utilizada na Computação e em outras áreas, é a operação inversa da exponencial. Existem 3 componentes num logaritmo: base, logaritmando e logaritmo, que são a, b e c do exemplo acima, respectivamente. Com os números reais a e b sendo positivos, a diferente de 0 e b = ac, assim o expoente c será o logaritmo.


Propriedades Operatórias do Logaritmo

loga(m * n) = logam + logan
logan/m = logan - loga
logamn = n * logam
log b n = Log a n Log a b para n > 0, b > 0, a > 0; b ≠ 1 e a ≠ 1
log a b = 1 log b a


Definição da Função Logarítmica

A função logarítmica tem como definição, o reverso de uma função exponencial. Por exemplo, uma função exponencial de base n invertida seria logn: IR*+ → IR. A relação entre um número real positivo x e um número real logax, será que x na base a será chamado de logaritmo, sendo sempre a > 0 e ≠ 1
