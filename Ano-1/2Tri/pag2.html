Funções Quadráticas

f(x) = ax2 + bx + c

Também chamada de Função do 2º Grau, ela possui uma forma de parábola em um gráfico, em que no máximo terá dois valores possíveis para x.


Parâmetros de Entrada

Parâmetro A

Este parâmetro define o quanto a concavidade da parábola é voltada para baixo ou para cima e também se é mais aberta ou fechada.

Se a > 0, a concavidade será voltada para cima.
Se a < 0, a concavidade será voltada para baixo.

Obs: Na função quadrática, é impossível o parâmetro a valer 0, pois assim, irá perder sua característica quadrática/polinomial e virar em uma função afim.

Parâmetro B

O parâmetro b indica como a função corta o eixo y (o eixo vertical), mostrando um qual lado do eixo y estará o vértice da parábola.

Se b > 0, a curva irá subir após cortar o eixo y .
Se b < 0, a curva irá descer após cortar o eixo y.
Se b = 0, é cortado o vértice exatamente no eixo y.

Parâmetro C

Irá definir com o parâmetro c, onde que a parábola vai cortar o eixo y, pois sendo x = 0 encontramos o valor de c.

f(x) = c, sendo x = 0 (onde corta o eixo y).


Valores do Vértice

v = ( - b 2 a , - Δ 4 a )
Estes são os respectivos valores do vértice em x e y. O vértice seria o ponto máximo (ou mínimo para crescentes) em que a concavidade da função chega.


Raízes da Função

Os valores de x serão quando a função encosta no eixo x. Dessa forma, a parábola pode encostar duas vezes, uma vez (quando fica no ponto máximo) e até mesmo não encostar.

Podemos descobrir as raízes da equação com da fórmula de Bhaskara, através do delta da função.

  • Se Δ > 0 → A função possui duas raízes distintas (dois pontos no eixo x).
  • Se Δ = 0 → A função possui uma única raiz (somente um ponto encosta, que é o vértice da parábola)
  • Se Δ < 0 → A função não possui raízes reais (a função não encosta no eixo x)
