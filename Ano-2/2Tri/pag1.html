Equações Lineares

São as equações em que sempre as incógnitas tem expoente 1 e não tem multiplicação e divisão entre elas. Assim, esse tipo de equação representa retas ou planos, e os pontos sendo os valores das incógnitas.

Fórmula Geral

a 1 x 1 + a 2 x 2 + a 3 x 3 + … + a n x n + b = 0

Sistemas Lineares

É o conjunto de equações lineares, que são interligadas através das mesmas incógnitas. Na computação, os sistemas lineares são usados para resolver problemas de otimização, modelar sistemas físicos e de redes, e em computação gráfica para manipular imagens e criar cenas 3D realistas.

Classficação dos Sistemas

Os Sistemas Lineares podem ser dividos em 3 tipos de solução: SPD, SPI e SI.

  • Possível: Possui pelo menos uma solução

    • Determinado: Somente uma solução. (SPD)
    • Indeterminado: Possui infinitas soluções possíveis. (SPI)

  • Impossível: Não possui solução exata para o sistema. (SI)


Sistemas 2x2

a 1 x + b 1 y = c 1 a 2 x + b 2 y = c 2

São os sistemas que possuem somente duas equações lineares, possuindo as mesmas incógnitas. No caso do exemplo acima, as incognítas são a, b, e c. Para solucionar esse tipo de Sistema, possui dois métodos para efetuar:

Método da Adição:

a 1 x + b 1 y = c 1 a 2 x + b 2 y = c 2 ⇒ Elimina-se uma incógnita: ⇒ y = c 1 a 2 - c 2 a 1 b 1 a 2 - b 2 a 1

Método da Substituição:

a 1 x + b 1 y = c 1 a 2 x + b 2 y = c 2 ⇒ Isola-se x = c 1 - b 1 y a 1 ⇒ substitui na segunda equação: ⇒ a 2 c 1 - b 1 y a 1 + b 2 y = c 2
