Graus e Radianos

O grau é uma unidade de medida de ângulo que é igual a 1/360 de uma volta completa. O radiano é uma unidade de medida de ângulo que é igual a 1/2π de uma volta completa.


Para converter graus para radianos, é necessário multiplicar o valor em graus por π/180. Para converter radianos para graus, é necessário multiplicar o valor em radianos por 180/π. Exemplo: 45° = π/4 radianos e 3π/4 radianos = 135°.


Para a computação, a conversão de graus para radianos é importante para a realização de cálculos envolvendo ângulos em funções trigonométricas, como seno, cosseno e tangente. Essas funções são essenciais em algoritmos de processamento de imagens, simulação de movimentos, análise de sinais e em diversas aplicações científicas e de engenharia.

Ciclo trigonométrico

É um Círculo Unitário (ou seja, com o valor de raio sendo 1) que demonstra de forma gráfica as razões trigonométricas. Dessa forma, é possível entender o comportamento dessas razões de forma mais simples através desse ciclo.


Ciclo Trigonométrico Interativo

Arcos Congruentes: São arcos que possuem valores diferentes, porém estão no mesma posição no ciclo trigonométrico.

Expressão Geral:

\[ \alpha + k\cdot 2\pi \qquad \text{ou} \qquad \alpha + k\cdot 360^\circ \qquad \text{com } k\in\mathbb{Z} \]

Primeira Determinação: É o menor ângulo possível que é côngruo a um determinado arco. Em outras palavras, é um valor que está entre 0 e 360 graus, pois é os valores possíveis em uma volta somente. Para obter a primeira determinação positiva, é necessário dividir o arco por 360° em uma divisão inteira.


Para ângulos que são negativos, é preciso dividir o valor absoluto do arco por 360°, o resto é a primeira determinação negativa. Para encontrar a positiva, se subtrai o valor absoluto do resto de 360°.
