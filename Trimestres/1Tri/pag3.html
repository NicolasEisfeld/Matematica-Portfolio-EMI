Sistema Octal

É a base composta por 8 algarismos diferentes, sendo eles: 0, 1, 2, 3 , 4, 5, 6 e 7

Ele foi uma alternativa ao sistema binários na informática e computação por um tempo, porém com o tempo essa posição ficou ao sistema hexadecimal (base 16) e o sistema binário ainda permanece como o principal sistema compacto computacional.

Destaque: Sempre uma base numérica será igual a quantidade de algarismos utilizados nela. Como sempre é necessário um algarismo para representar o vazio ou neutro, o algarismo maior do sistema sempre vai ser o próprio número da base menos 1.

Conversão de Decimal para Octal

É necessário fazer a divisão do número no sistema decimal por 8 e guardar o resultado da divisão. Em seguinte, repita a divisão do resultado da primeira divisão por 8 até não houver mais restos. Assim, os restos das divisões efetuadas de trás para frente será o valor decimal em octal.

Exemplo:ㅤ18 / 8 = 2, resto 2;

Conversão de Octal para Decimal

Cada dígito da base Octal deve ser multiplicado pelas potências de base 8. Após, deve somar os resultados e assim, terá o valor no sistema decimal.

Exemplo:ㅤ(307)8 = 3 x 82 + 0 x 81 + 7 x 80;
